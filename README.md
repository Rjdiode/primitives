# Primitives

Trackpad-first drafting canvas with snap grid, rulers, angled guides, and geometric primitives.

## Quick start

```bash
python3 -m http.server 8080
```

Open http://localhost:8080 — or open `index.html` directly.

## Features

- Snap grid and object snap (centers, edges, vertices)
- Horizontal / vertical guides from rulers
- Protractor tool: place grid anchors and angled guides (15° steps)
- Line and stroke tools between guide crossings
- Rectangle, square, ellipse, circle
- Marquee select, pan (Space / middle-click), pinch zoom
